A Demo for messaging in WebRTC

Run two or more instances of this webpage along with the server to test this application.
In the first field give your own username, and in the second field give the peers user names in a comma separated list.
Press initialize on all webpages.
reate an offer on the pages until all "know each other", and then send the message.
Check the browser console to see the output.

This application is intentionally made simple to avoid cluttering with non WebRTC related code.
